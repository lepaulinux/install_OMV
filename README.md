# install_OMV

Install Open Media Vault on Raspberry Pi 4

## Steps

1. Create the SD card  
2. Update the OS and change the password  
3. Install Open Media Vault  
4. Change the General Settings  
   - Enable Auto Login  
   - Change the Password  
   - Change the port number (if necessary)  
5. Set the Time Zone  
6. Set the Hostname  
7. Configure the Network IP address  
8. Update Management  
   - Install updates  
9. Install Plugin  
   - Fail2Ban plugin  
10. Disks - Format  
11. File System - Add the partition  
12. Shared Folders - Create a folder to be shared  
13. SMB - Windows file share  
14. OMV-Extras - Install
